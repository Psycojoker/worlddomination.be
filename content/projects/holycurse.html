Commandes

Main view mode

  • q exit
  • j/k move up/down (up and down will work to but not in the good way)
  • space toggle the current mission
    If the mission have a question, a new mission description will be ask after the toggling (enter nothing to avoid creating a new mission)
  • a append a new mission to the current realm (broken on the "in X time" pseudo-realms, will just take the realm of the current todo)
  • n add a new misison to the default realm
  • d remove the current mission (warning: NO UNDO)
  • r rename the current mission
  • c add a new completed mission (ie: "I've done that and I want to add it for historic")
  • C move current mission to a new realm
  • R toggle the current mission and recreate one with the same description
  • + tickle a mission for one day
  • = tickle a mission for 3 hours
  • t set the due date of the mission to today
  • T set the due date of the mission for in 3 days
  • w set the due date of the mission for in one week
  • W remove any due date one the current mission
  • p add a quest to the current mission (if a quest with a same description exist, no new quest will be created)
  • s "swap". Create a new quest with the mission description and ask for the description of a new mission for this quest
  • 2 move to the realm view
  • : will enter command line (useless for the moment)

Realm view

  • q exit
  • 1 move to the main view
  • j/k move up/down (up and down will work to but not in the good way)
  • J/K move the position of the current realm up/down
  • space toggle if a realm if hidden
  • a/n add a new realm
  • r rename the current realm
  • d delete the current realm. WARNING will not work if the realm still has visible todos. All the ancien todos will me moved to the default realm. You can't remove the default realm

In user input

  • enter finish the user input
  • most of the time, entering nothing will result in annulation of the current action

In chosing a realm mode (the C command in main view)

  • q return to the main view (do nothing)
  • j/k move up/down (up and down will work to but not in the good way)
  • enter select the current realm
