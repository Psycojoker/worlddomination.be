Softwares

  • HolyGrail: a very high level library to manage the holy quest of your life ! (In other words it's an high level library to manage a next actions list in the GTD's way).
  • HolyCurse: HolyView is a ncurse interface to HolyGrail.
  • HolyView: HolyView is a ncurse tool design for long term organisation. It's purpose is to help you chose what to do now to be the most effective in the long term.
  • ULR: a minimalist web browser who read a list of urls in a file. Plan to be used with newsbeuter and/or rsstail.
  • PipeToolKit: a collection of small scripts that modify input received via shell pipe

Tools and scripts

  • HowTo script a irc bot in the unix way: highlight configurable bot using ii, rsstail, bti, twurl and others. Also include a way to pipe a RSS to a identi.ca/twitter account.
  • puniq, eliminate duplication from a continuous stream (ie: tail -f, rsstail). No, | sort | uniq won't work on those.
  • rssator, create your own rss-irc chan.
  • pytestator, a vim-script to facilitate the coding of unittest in python.
  • mobifyer, transform the core text of a web page into a .mobi file (works also for epub) using the algorithm of redability and ebook-convert (from calibre).
