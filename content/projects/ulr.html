Présentation

Le principe est simple: vous rajoutez des urls dans un fichier (ou les passé en argument à ULR, il les rajoutera lui même). Puis vous le lancez le browser qui naviguera dans cette liste d'urls.

Screenshot

Raccourcis

  • esc/q: quitter sans modifier la liste
  • n/espace: passe à l'url suivante (espace fait baisser l'écran aussi, j'ai eu la flemme de corriger ce bug)
  • r: fait un refresh de la page courante
  • s: sauve l'avancement dans la liste des urls (en gros supprime du fichier urls les urls déjà vu)
  • S: pareil puis quitte
  • y: copie l'url courante dans le clipboard de X (il faut avoir xclip installé)
  • f: envoie l'url vers firefox (il faut bien entendu qu'il soit installé)
  • B: enregistre l'url courante dans un fichier nomé "bookmarks"

Get the code

git clone git://git.worlddomination.be/python/ulr.git

Remarques

  • Conçu pour être utilisé comme « browser » dans newsbeuter
  • Rien ne vous empêche de combiner ULR avec rsstail ou n'importe quel autre script ou programme

Config pour newsbeuter

Simplement rajouter ulr comme browser dans le fichier de config de newsbeuter. Quand vous lancerez le browser (o) cela rajoutera l'url à la liste des urls à lire, puis après il faudra lancer ulr manuelement pour pouvoir parcourrir cette liste d'urls.
